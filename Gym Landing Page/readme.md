# Gym Landing Page

## Overview
A modern, responsive gym landing page built with HTML and CSS.

## Features
- Responsive design for mobile and desktop
- Clean, professional layout
- Smooth navigation
- Image galleries
- Contact information

## Project Structure
```
Gym Landing Page/
├── index.html
├── style.css
├── images/
└── readme.md
```

## Installation
1. Clone or download the project
2. Open `index.html` in your browser
3. No dependencies required

## Usage
- Customize the content in `index.html`
- Modify colors and styling in `style.css`
- Replace images in the `images/` folder
