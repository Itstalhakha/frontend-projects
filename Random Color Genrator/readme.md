
# Random Color Generator

A simple tool to generate random colors with various formats and options.

## Features

- Generate random hex colors
- Generate random RGB colors
- Copy to clipboard functionality
- Real-time color preview

## Usage

1. Click the "Generate" button to create a new random color
2. View the color preview in real-time
3. Copy the color code in your preferred format (Hex)
